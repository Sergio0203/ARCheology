# ARCheology

**ARCheology** é um projeto desenvolvido na trilha de desenvolvimento AR/VR na NEXVISUAL.

## Funcionalidades Pessoais Adicionadas

- **Efeito de Rotação Orgânica (Tumbling):** Um objeto selecionado não escaneado fica rotacionando em torno de si mesmo em diversas direções.
- **Icons Adicionados:** Permite que seja definido um icon que aparece no canvas após ser selecionado para objeto
- **Feedback Sonoro:** O scanner faz um barulho de scanner quando estar scaneando um objeto.
- **Texto do Scanner Desativado:** Agora após retirar o objeto do scanner a mensagem de Scan finalizado é desativada. 
- **Recentralização do objeto:** Se um objeto não estiver na sua rotação centralizada e ele ja tiver sido escaneado ele volta a sua rotação centralizada.
